# Currency Converter 💱

Простое и удобное GUI-приложение для конвертации валют с сохранением истории операций.
*Язык:* Python.
*Библиотеки:* Tkinter (стандартная), Requests.
*API:* ExchangeRate-API.

## 🛠️ Установка и запуск

**1. Установите Python**
Нужна версия **3.8 или новее**.
> *Tkinter уже встроен в Python.*
**2. Получите API-ключ**
Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com/), зарегистрируйтесь и получите бесплатный ключ.
**3. Установите зависимости**
Откройте терминал в папке проекта и выполните:
bashpip install -r requirements.txt

Копировать
**4. Запустите приложение**
bashpython currency_converter.py

Копировать
Откроется окно программы.

## ⚙️ Как вставить API-ключ

1. Откройте файл `currency_converter.py`.
2. Найдите строку:
   ```python
   API_KEY = 'ВАШ_API_КЛЮЧ'
   ```
3. Замените `'ВАШ_API_КЛЮЧ'` на ключ, который вы получили на сайте.
4. Сохраните файл.
> *Если ключ не указан или введён неверно, программа покажет ошибку при запуске.*

## ✅ Как использовать приложение

1. Выберите валюту «Из» и «В».
2. Введите сумму (только положительное число).
3. Нажмите кнопку «Конвертировать».
4. Результат появится под кнопкой.
5. Все операции сохраняются в таблицу и в файл `history.json`.
6. История сохраняется даже после закрытия программы.
